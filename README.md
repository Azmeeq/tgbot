# 📚 Student Bot - Бот для управления домашними заданиями

Telegram-бот для группы студентов, позволяющий управлять домашними заданиями и личными задачами.

## 🚀 Возможности

### Для группы (групповой чат):
- ✅ Добавление общих домашек
- 📋 Просмотр всех домашек
- 🔍 Фильтрация по предметам
- ✅ Отметка выполнения
- 🗑️ Удаление (автор или админ)
- 📊 Статистика группы

### Для личного использования (ЛС):
- 📝 Личные задачи
- ⏰ Дедлайны
- ✅ Отметка выполнения
